# Index

This repository contains the following key files:

- `app/main.py` - FastAPI sentiment prediction API.
- `app/logging_config.py` - Logging utilities for prediction and performance tracking.
- `frontend/app.py` - Streamlit user interface.
- `model/bertweet_sentiment/` - Local BERTweet model files.
- `Dockerfile` - Docker build configuration.
- `docker-compose.yml` - Container orchestration.
- `requirements.txt` - Python dependencies.
- `README.md` - Project overview and usage.
